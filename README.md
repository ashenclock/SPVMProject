# 🧭 SPVM Project

Software Engineering university project for workforce and shift management in a citizen-services company scenario.

## 👥 Team

- [Slenderman0039](https://github.com/Slenderman0039)
- [jackaljo](https://github.com/jackaljo)
- [ashenclock](https://github.com/ashenclock)
- [JovenLD](https://github.com/JovenLD)

## 🎯 Project Goal

Design and implement a system to manage:

- employee profiles and roles
- shift planning and service coverage
- absences, substitutions, and constraints
- salary-related operational data

## 🛠️ Stack

- Java / JavaFX
- Maven
- MariaDB JDBC
- CalendarFX
- Javax Mail

## 📁 Repository Content

- `SPVMProject_CODE/`: JavaFX codebase
- `RAD.pdf`, `SDD.pdf`, `ODD.pdf`: software engineering documentation
- `mockup/`: UI/UX mockups

## 🚀 Build & Run

From `SPVMProject_CODE/`:

```bash
./mvnw clean package
./mvnw javafx:run
```

(On Windows use `mvnw.cmd`.)
